# Siteportal

## Project Overview
SitePortal is a web and mobile-based platform developed for the CSE343 Software Engineering course. 
The project focuses on designing and deploying a cloud-based system that operates independently of the developerâ€™s local environment.

## Deployment & Infrastructure
- Deployed on a DigitalOcean cloud virtual machine
- Uses managed PostgreSQL (Supabase) as a cloud database
- Environment variables are configured at the server level
- Accessible via a custom domain: https://siteportal.com.tr

## Technology Stack
- Backend: Node.js (Express.js)
- Web Frontend: HTML, CSS, JavaScript
- Mobile Application: Flutter
- Database: Supabase (PostgreSQL)
- Hosting: DigitalOcean
